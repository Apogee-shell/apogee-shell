ФИЗИКА УДАРНЫХ ВОЗДЕЙСТВИЙ

Мы изучаем поведение гетерогенных структур при критических динамических нагрузках. Наша цель — оптимизация межфазного взаимодействия для предотвращения расслоения материалов.

02 // TECHNICAL REQUEST

МАТЕРИАЛЫ ДЛЯ АПРОБАЦИИ

СВМПЭ (UHMWPE)
Однонаправленные (UD) листы, плотность 130–160 г/м², для формирования антитравматического слоя.
Керамические элементы
Монолитные или мозаичные панели на основе Al₂O₃ (оксид алюминия) чистотой не менее 98%.
Смоляные системы
Низковязкие эпоксидные и полиуретановые системы с высокой скоростью полимеризации.
Арамидные ткани
Ткани полотняного переплетения (Ballistic Grade) для усиления тыльного слоя.
